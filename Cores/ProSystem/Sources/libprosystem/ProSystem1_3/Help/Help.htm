ProSystem Emulator Documentation

Help
The help menu allows you to view the online documentation for the ProSystem emulator and view the about dialog box.

Finding help
You can view the online help by going to the help menu and selecting the contents or index menu items or use the shortcut F1. The help window will open and will allow you to view the user's guide.

About ProSystem emulator
You can view the about dialog box by going to the help menu and selecting the about menu item.
